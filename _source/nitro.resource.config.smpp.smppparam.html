clientmode¶

Mode in which the client binds to the ADC. Applicable settings function as follows: * TRANSCEIVER - Client can send and receive messages to and from the message center. * TRANSMITTERONLY - Client can only send messages. * RECEIVERONLY - Client can only receive messages.<br/>Default value: TRANSCEIVER<br/>Possible values = TRANSCEIVER, TRANSMITTERONLY, RECEIVERONLY.
