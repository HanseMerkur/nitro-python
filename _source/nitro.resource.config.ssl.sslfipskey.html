fipskeyname¶

Name for the FIPS key. Must begin with an ASCII alphanumeric or underscore (_) character, and must contain only ASCII alphanumeric, underscore, hash (#), period (.), space, colon (:), at (@), equals (=), and hyphen (-) characters. Cannot be changed after the FIPS key is created. The following requirement applies only to the NetScaler CLI: If the name includes one or more spaces, enclose the name in double or single quotation marks (for example, “my fipskey” or ‘my fipskey’).<br/>Minimum length = 1.

inform¶

Input format of the key file. Available formats are: SIM - Secure Information Management; select when importing a FIPS key. If the external FIPS key is encrypted, first decrypt it, and then import it. PEM - Privacy Enhanced Mail; select when importing a non-FIPS key. <br/>Default value: SIM<br/>Possible values = SIM, DER, PEM.

iv¶

Initialization Vector (IV) to use for importing the key. Required for importing a non-FIPS key.<br/>Minimum length = 1.

key¶

Name of and, optionally, path to the key file to be imported. /nsconfig/ssl/ is the default path.<br/>Minimum length = 1.

modulus¶

Modulus, in multiples of 64, of the FIPS key to be created.<br/>Maximum length = 4096.

size¶

Size.

wrapkeyname¶

Name of the wrap key to use for importing the key. Required for importing a non-FIPS key.<br/>Minimum length = 1.
