action¶

Action to perform on the outgoing IPv6 packets that match the PBR6. Available settings function as follows: * ALLOW - The NetScaler appliance sends the packet to the designated next-hop router. * DENY - The NetScaler appliance applies the routing table for normal destination-based routing.<br/>Possible values = ALLOW, DENY.
